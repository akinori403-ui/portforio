Cafe Website

Calm Cafe

落ち着いた雰囲気を大切にした、カフェのWebサイトです。

Overview

コーヒーとスイーツを楽しめる落ち着いた雰囲気のカフェをイメージして制作した架空サイトです。 ナチュラルな配色と余白を活かし、訪問者がリラックスして閲覧できるよう設計しました。

使用技術

HTML / CSS / JavaScript

制作期間

約1週間

担当範囲

デザイン / コーディング

工夫した点

カフェの「ゆったりした時間」を伝えるため、余白を広めに設定しました。 温かみのある配色で、落ち着いた印象になるよう意識しています。

苦労した点

メニュー一覧の見せ方や、スマートフォン表示時の余白調整に苦労しました。 最終的にシンプルな構成に整えました。
